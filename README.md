# Genetic Disease Classification System 🧬

A web-based database prototype designed for the classification and analysis of genetic diseases. 

> **Status:** Work in Progress (WIP) - Phase 1 🚧

### 🎯 Project Purpose
The primary goal of this project is to build a structured, accessible web database that categorizes genetic disease data, making it easier to filter and analyze.

### 💻 Tech Stack (Planned & Current)
*   **Frontend:** HTML5, CSS3 (Current UI Prototypes)
*   **Backend:** Python (Planned for logic and API)
*   **Database:** SQL (Planned for data storage)

### 📊 Current Progress
Currently, the project is in its initial phase. The `UI-Prototype` folder contains the early frontend mockups. Data set integration and the backend architecture are currently under development.
